# 静态文件下载站

基于 GitHub Pages 构建的静态文件托管与下载页面，无需服务器，完全免费。

## 项目特点

- 纯静态部署，托管于 GitHub Pages
- 无需后端，文件列表通过 `files.json` 定义
- 支持文件搜索与多维度排序
- 响应式设计，适配移动端与桌面端
- 部署简单，更新方便

## 注意事项
- GitHub Pages 有仓库容量（1GB）与流量限制
- 单文件不可超过25MB
- 更新文件后需同步修改 files.json

## 使用方法

1. 将文件上传至仓库的 `files/` 目录
2. 更新 `files.json` 文件，添加文件信息（名称、路径、大小、描述等）
3. 推送至 GitHub 仓库即可自动部署
